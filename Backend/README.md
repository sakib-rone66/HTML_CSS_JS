# 🚀 Backend Development Journey (Node.js, Express.js & REST API)

Welcome to my backend learning repository! This repository documents my journey of learning backend development using **Node.js**, **Express.js**, and **REST APIs**. The goal is to build a strong foundation in server-side development, API design, database integration, and authentication.
